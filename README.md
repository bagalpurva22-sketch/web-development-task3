# CSS Selectors & Styling Assignment


### link Stylesheet
* Connect the `style.css` file to your `index.html` file using the `<link>` tag inside the `<head>` section.

### Base Body Background
*  background color for the `<body>` element is orangered.

###  Heading Elements
*  **3 different heading tags** (e.g., `<h1>`, `<h2>`, `<h3>`) and assigned them all a class name of `head` (`class="head"`).

###  Class-based Text Color
*  apply a **dark blue** text color to all elements containing the class `head`.

### targeted Heading Background
* Apply a specific background color to the **second heading element** using its unique ID selector: `id="bglime"`.

###  Specific Text Overriding
* Set the text color of the `"head"` class inside the **2nd heading** to **red**.


###  Specific Paragraph Customization
* Customize the **second paragraph** using an ID selector `id="para"` to make its text color **green**.

### Specific Overrides & Sizing
*  setting 'paragraph'text color in 2nd paragraph is **black** using span
